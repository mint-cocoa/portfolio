안녕하세요, 네트워크 프로그래밍과 서버 개발에 관심을 가지고 있는 신입 지원자 배진후입니다.

학부 졸업 작품으로 웹 서버를 개발하면서 백엔드에 흥미를 갖게 되었습니다. 다수의 사용자가 동시에 접속하는 상황을 고려해 부하 테스트를 수행하는 과정에서 성능 최적화와 안정성에 대한 관심이 생겼고, 단순한 요청-응답 구조를 넘어 실시간 동기화와 상태 관리, 지연 시간 최소화 같은 복합적인 과제를 다루는 게임 서버 개발에 매력을 느끼게 되었습니다.

졸업 작품에서는 예약, 티켓 발급, 시설 관리, 실시간 위치 추적, 알림 등 테마파크의 다양한 서비스를 독립적인 MSA로 분리해 개발했습니다. ArgoCD와 GitHub Actions를 연동해 코드 변경이 Git에 반영되면 자동으로 빌드 후 개발용 쿠버네티스 클러스터에 배포되도록 구성해 팀원 간 환경 차이로 인한 호환성 문제를 줄였고, 백엔드는 FastAPI로 구현해 Swagger를 통해 API 문서를 자동화함으로써 팀 간 연동을 원활하게 만들었습니다.

소켓 프로그래밍에 대한 이해를 높이기 위해 Linux 환경에서 epoll과 io_uring을 활용한 에코 서버를 직접 구현하고, 두 I/O 모델의 성능을 비교 분석하는 프로젝트도 진행했습니다. 클라이언트-서버 간 통신 구조 설계부터 소켓 래핑 클래스까지 직접 구현했으며, 처리량을 높이기 위해 CPU 코어별로 독립적인 I/O 인스턴스를 배치하는 멀티스레딩 방식을 적용했습니다. 벤치마크 코드를 수정해 직접 테스트를 수행했고, 측정 결과를 바탕으로 각 모델의 장단점과 실제 활용 시 고려할 요소들을 구체적으로 이해할 수 있었습니다.

이러한 경험들이 기술적인 역량 외에 협업과 커뮤니케이션의 중요성도 함께 인식하게 해주었습니다. 앞으로도 복잡한 문제를 기술적으로 해결하면서 팀과 긴밀히 협력해 더 나은 서비스를 만들어 나가는 개발자가 되겠습니다. 부족한 실무 경험은 빠르게 채워나가며 맡은 업무에 책임감을 가지고 성실히 임하겠습니다. 감사합니다.
